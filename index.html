Cypher Academy Road Map

Block One
HTML & CSS

Learn the building blocks of Web Development. Gain a thorough grasp of HTML and CSS by learning how to write reusable and scalable code through CSS architecture and the BEM model.

Block Two
JavaScript

Start your foundation in object-oriented computer programming with JavaScript. Bring your web applications to life by building fun and meaningful applications!

Block Three
Advanced JavaScript

Understand advance concepts such as scoping, function expressions, constructors, objects, objects literals, and improve your ability to debug problems. Build large scale web applications using APIs and learn how to weave through JSON data.

Block Four
ReactJS

Get an introduction in the fundamentals of React, JSX, props, states and eventing. Go past vanilla JavaScript with a bigger tool set in JavaScript libraries.

Block Five
NodeJS

Grasp NodeJS concepts, package management, and using the your terminal.

Block Six
WordPress

Jump into the top-leading CMS (Content Management System) on the market. Build and customize themes, templates, plugins, tags, and hooks resulting with a beautiful web application that you can use.
